# Phase Plan Index

## Execution Protocol
1. Execute one task at a time within the active phase.
2. Validate the completed task against its Done and verification criteria.
3. Move to the next task only after validation passes.
4. Commit and push each verified task to the configured remote.
5. If validation fails, stop progression, fix, and re-test before continuing.
